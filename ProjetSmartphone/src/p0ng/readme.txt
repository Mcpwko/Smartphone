B. Running the Program
----------------------

  Note: You need to install the Java Runtime Environment.

  To run the game, type on the terminal:
  
    java GamePanel

----------------------
C. Controls
----------------------

  +-----------------------------------------+
  | Keys             Action                 |
  +-----------------------------------------+
  | P                Pauses the game        |
  | Up/Down Arrows   Moves the right paddle |
  | A/Z              Moves the left paddle  |
  +-----------------------------------------+
